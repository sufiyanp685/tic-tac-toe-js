# ❌⭕ Tic-Tac-Toe Game (JavaScript)

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This project demonstrates core JavaScript concepts like DOM manipulation, event handling, and game logic.

---

## 🚀 Features

- ❌⭕ Two-player gameplay (X vs O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔁 Restart / Reset game option
- 🎯 Turn indicator
- ⚡ Smooth and responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---


## 💡 What I Learned

- DOM manipulation using JavaScript
- Event handling (click events)
- Game logic implementation
- Conditional rendering
- Writing clean and structured code
